Fish-AIR Dataset is available through RESTful JSON Fish-AIR API. The base URL for v1 is:https://fishair.org/api/.

Fish-AIR Dataset is under CC BY-NC license. Please check the license information for multimedia objects within the metadata separately.

The API key is a unique identifier that authenticates requests of calling the API. Without a valid 'x-api-key' in request header, your request will not be processed. Please contact Xiaojun Wang at xwang48@tulane.edu for a API key. Please DON'T share your api key with others.

Please check Fish-AIR API Documentation for further information
